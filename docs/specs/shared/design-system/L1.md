# Accessibility & Responsive Design (DSGN) — L1 High-Level Requirements

Cross-cutting user-interface guarantees: accessible form validation, text contrast, and responsive layout across viewports.

Part of the subsystem-organized specification structure — see the [specification registry](../../README.md) for subsystem ownership, conventions, the ID scheme, and the legacy ID mapping. Detailed requirements live in [L2.md](L2.md); each L2 traces to exactly one L1 in this file.

---

## L1-DSGN-017: Accessibility

Interactive flows shall be operable with assistive technology: form errors shall be programmatically associated and focus-managed, and text shall meet contrast requirements.

---

## L1-DSGN-018: Responsive design

All pages shall function across viewport sizes from extra-small (<576px) through extra-large (≥1200px), with a mobile navigation pattern on narrow screens.
