# Notifications (NOTIF) — L1 High-Level Requirements

Realtime in-app notifications for connection, match, message, and meeting events, with read tracking and deep links.

Part of the capability-based specification structure — see the [capability registry](../README.md) for conventions, the ID scheme, and the legacy ID mapping. Detailed requirements live in [L2.md](L2.md); each L2 traces to exactly one L1 in this file.

---

## L1-NOTIF-011: In-app notifications

Users shall receive realtime in-app notifications for connection, match, message, and meeting events, mark them read, and navigate from a notification to the relevant page.
