# Regional Map & Cohorts (COHRT) — L1 High-Level Requirements

Consent-aware regional aggregation of eligible participants and cohort-readiness display.

Part of the subsystem-organized specification structure — see the [specification registry](../README.md) for subsystem ownership, conventions, the ID scheme, and the legacy ID mapping. Detailed requirements live in [L2.md](L2.md); each L2 traces to exactly one L1 in this file.

---

## L1-COHRT-012: Regional map and cohorts

Users shall see aggregate regional participation counts limited to eligible, consenting participants, with regions flagged when enough participants exist to form a cohort.
