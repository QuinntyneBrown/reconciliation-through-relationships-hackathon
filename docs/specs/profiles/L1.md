# Participant Profiles (PROF) — L1 High-Level Requirements

Shareable participant profiles, the own-profile shortcut, and the profile-based connection entry point.

Part of the capability-based specification structure — see the [capability registry](../README.md) for conventions, the ID scheme, and the legacy ID mapping. Detailed requirements live in [L2.md](L2.md); each L2 traces to exactly one L1 in this file.

---

## L1-PROF-008: Participant profiles

Participants shall view each other's profiles (facilitator profiles excluded), reach their own profile via a shortcut, and initiate or manage a connection from a profile.
