# Portal Shell & Navigation (SHELL) — L1 High-Level Requirements

The authenticated portal chrome: role-appropriate primary navigation with current-page indication, and the account menu.

Part of the capability-based specification structure — see the [capability registry](../README.md) for conventions, the ID scheme, and the legacy ID mapping. Detailed requirements live in [L2.md](L2.md); each L2 traces to exactly one L1 in this file.

---

## L1-SHELL-004: Portal navigation and account menu

Authenticated users shall navigate between their role's destinations via a primary navigation with current-page indication, and shall reach their own profile and sign out via an account menu.
