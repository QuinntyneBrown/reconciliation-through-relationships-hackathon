# Connections & Messaging (CONN) — L1 High-Level Requirements

Mutual-consent one-to-one connections with realtime chat, including facilitator review when auto-matching is disabled.

Part of the subsystem-organized specification structure — see the [specification registry](../README.md) for subsystem ownership, conventions, the ID scheme, and the legacy ID mapping. Detailed requirements live in [L2.md](L2.md); each L2 traces to exactly one L1 in this file.

## L1-CONN-009: Connections and messaging

Participants shall request, cancel, accept, and conduct one-to-one connections with realtime chat, gated by mutual consent and, when auto-matching is disabled, by facilitator review.
