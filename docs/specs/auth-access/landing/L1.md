# Public Landing (LAND) — L1 High-Level Requirements

The public, unauthenticated surface: program explanation, testimonials, and the routes into sign-up and sign-in.

Part of the subsystem-organized specification structure — see the [specification registry](../../README.md) for subsystem ownership, conventions, the ID scheme, and the legacy ID mapping. Detailed requirements live in [L2.md](L2.md); each L2 traces to exactly one L1 in this file.

---

## L1-LAND-001: Public landing page

Anonymous visitors shall learn the program's purpose, four-step journey, and participant testimonials, and shall be routed to account creation or sign-in.
